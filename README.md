# 计算机硕博扫盲手册

本指南由SecureNexusLab多位团队成员共同编写，科研本身包含了太多细节，这篇指南书写的初衷是希望能够给计算机领域新入门的硕博提供科研方面的扫盲。按照形成一篇论文的过程作为行文依据，包括文献调研、论文阅读、论文复现、论文Idea、实验环境、论文写作与润色、论文绘图、论文投稿、科研写作。编写成员来自山东大学、浙江大学、北京邮电大学、西安电子科技大学、江西师范大学，同时也收到了多位师傅的建议，在此对各位表达感谢。也希望读到手册的各位可以不吝赐教，对手册提出新的建议。
